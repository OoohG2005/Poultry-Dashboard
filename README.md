# 🐔 Smart Poultry House Monitoring System

## AI-Powered Environmental Monitoring & Disease Prediction

### Features
- Real-time monitoring of temperature, humidity, ammonia, CO2
- AI disease risk prediction
- Automated alerts and recommendations
- Interactive data visualization

### Quick Start
1. Install dependencies: `pip install -r requirements.txt`
2. Run dashboard: `python run.py`
3. Open browser: http://localhost:8501
